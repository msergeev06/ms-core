1. На странице подключается файл /msergeev_config.php
	1.1. В нем определяются основные настройки Базы данных и путей к файлам
	1.2. В нем также подключается основной файл библиотеки /msergeev/include.php
		1.2.1. В файле библиотеки поключается файл подгрузки инструментов ядра /msergeev/core/tools.php
			1.2.1.1. В файле подгрузки инструментов ядра подключаются все необходимые инструменты, расположенные в
						папке /msergeev/core/tools/
		1.2.2. В файле библиотеки также подключается основной инициализирующий файл ядра /msergeev/core/core.php
			1.2.2.1. Файл ядра подключает библиотеку работы с Базой Данных и инициализирует подключение к DB (DataBase)
			1.2.2.2. Файл ядра подключает библиотеку работы с Опциями и инициализирует ее (Options)
			1.2.2.3. Файл ядра подключает библиотеку Загрузчика и инициализирут его (Loader)
				1.2.2.4. При инициализации Загрузчик пробегается по каталогу пакетов /msergeev/packages/
							и сохраняет себе список установленных пакетов.
			1.2.2.4. Файл ядра подключает библиотеку работы с Пользователями и инициализирует ее (Users)
			1.2.2.5. Файл ядра подключает файлы для обработки Исключений (Exception)
			1.2.2.6. Файл ядра подключает все оставшиеся библиотеки ядра из каталога /msergeev/core/lib
			1.2.2.7. Файл ядра инициализирует библиотеку Локализации (Loc)
			1.2.2.8. Файл ядра подключает файлы описания Сущностей (Entity)
			1.2.2.9. Файл ядра подключает файлы описания Таблиц базы данных (Tables)
2. После подключения основного конфигурационного файла можно работать со всеми функциями ядра в любой момент.
	Использование функций ядра осуществляется с использованием namespace.
3. Для использования функций из дополнительных пакетов - необходимо подключить требуемый пакет с помощью функции
	MSergeev\Core\Lib\Loader::IncludePackage("название_пакета");
	3.1. При подключении нового пакета Загрузчик проверяет указан ли пакет, установлен ли пакет и не был ли он
		подключен ранее.
	3.2. Если все условия подключения успешны, Загрузчик осуществляет следующие действия:
		3.2.1. Проверяет, существуют ли у пакета зависимости с другими пакетами. Подобные зависимости описываются в
				файле /msergeev/packages/[имя_пакета]/required.php. В этом файле присутвуют два массива:
				$arRequiredPackages и $arAdditionalPackages. $arRequiredPackages описывает ОБЯЗАТЕЛЬНЫЕ пакеты,
				без которых данный пакет работать не будет. $arAdditionalPackages описывает ДОПОЛНИТЕЛЬНЫЕ пакеты,
				которые расширят функционал данного пакета.
			3.2.1.1. Если у пакета существуют зависимости с другими пакетами, Загрузчик подключает файл зависимостей
						для изучения
				3.2.1.1.1. Если массив $arRequiredPackages не пустой бежит по нему и проверяет, установлены ли
							обязательные пакеты. Если установлены и не были еще подключены - вызывает сам себя для
							подключения пакета (с этим пакетом также будут проведены проверки зависимостей). Если же
							требуемый обязательный пакет не установлен - умирает с ошибкой: Необходимо установить
							обязательный пакет [имя_пакета]
				3.2.1.1.2. Если массив $arAdditionalPackages не пустой бежит по нему и проверяет, установлены ли
							дополнительные пакета. Если установлены и не были еще подключены - вызывает сам себя для
							подключения пакета (с этим пакетом также будут проведены проверки зависимостей). Если же
							требуемый дополнительный пакет не установлен - ничего не делает.
		3.2.2. Подключает основной файл пакета /msergeev/packages/[имя_пакета]/include.php
		3.2.3. Проверяет, есть ли у пакета файл Опций-по-умолчанию /msergeev/packages/[имя_пакета]/default_options.php.
				Если файл есть, подключает его и загружает опции, при помощи библиотени Опций (Options)
			3.2.3.1. Также если в опциях-по-умолчанию указано имя шаблона (TEMPLATE) - устанавливает этот шаблон для
						вывода в публичной части. Если файла опций нет, либо нет данной опции - устанавливается шаблон
						по-умолчанию .default
		3.2.4. Загружает языковые файлы для пакета, при помощи библиотеки Loc
		3.2.5. Также сохраняет следующие параметры для всех пакетов (данные параметры будут доступны позднее из функций
				загрузчика):
			3.2.5.1. Путь к публичной директории (Loader::getPublic("имя_пакета"))
			3.2.5.2. Относительный путь к публичной директории (Loader::getSitePublic("имя_пакета"))
			3.2.5.3. Путь к загружаемым файлам пакета (Loader::getUpload("имя_пакета"))
			3.2.5.4. Путь к действующему шаблону пакета (см п.п. 3.2.3.1) (Loader::getTemplate("имя_пакета"))
			3.2.5.5. Относительный путь к действующему шаблону пакета (Loader::getSiteTemplate("имя_пакета"))
		3.2.6. Устанавливает флаг успешной загрузки пакета.
